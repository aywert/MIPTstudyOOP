## Боты

### Silly Bot (Глупый бот)

**Стратегия:** Жадный алгоритм с примитивной безопасностью

#### Алгоритм:

1. **Поиск еды**: Находит ближайшего кролика (по евклидову расстоянию)
2. **Движение**: Жадно двигается к кролику (сначала по X, потом по Y)
3. **Безопасность**: 
   - Проверяет, не врежется ли в стену или другую змею
   - При опасности выбирает безопасное направление в порядке приоритета:
     - Прямо → налево → направо
   - Запрещает разворот на 180 градусов
4. **Тупик**: Если все направления опасны, продолжает движение прямо


### Smart Bot (Умный бот)

**Стратегия:** Анализ тепловой карты (Heatmap-based) с мультифакторной оценкой каждой клетки

#### Основная идея

Умный бот не просто движется к еде, а оценивает каждую клетку игрового поля, учитывая множество факторов:
- Привлекательность еды
- Опасность столкновения с врагами
- Риск врезаться в собственное тело
- Близость к стенам

От каждого яблока/врага/стены вносится вклад в тепловую карту
```
for each x:
  for each y:
    dist = (x-x0)^2 + (y-y0)^2
    heatmap[x][y] += 500.0 / (dist + 1.0);
```

## feature
🌀 Портал - уникальная фишка игры

### Что делает мою змейку особенной?

В то время как классические реализации змейки ограничиваются базовой механикой "съешь еду - стань длиннее", мой проект добавляет новый элемент - **порталы**, которые кардинально меняют игровой процесс и открывают новые тактические возможности.

### В чем суть порталов?

Порталы - это парные телепорты, расположенные в случайных безопасных позициях на карте. Когда голова змеи попадает в один портал, она мгновенно телепортируется в другой.
 
Она добавляет элемент неожиданности в игру: 
 - Соперник может неожидать резкого появления змеи у себя под носом. 
- Позволяет сокращать путь к кроликам
